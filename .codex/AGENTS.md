# Project Description
- Project Name: CGI Reflect
- Description: An HR Information System whose main use case are performance reviews, feedbacks and surveys.

# Architecture
- Frontend
    - React
- Backend
    - Python : FastAPI
- Infra as a Code
    - AWS CDK in Typescript

## Coding Instructions
- Ensure that you follow good coding standards like consistent naming conventions and the use of comments.
- Write Unit Test whenever possible and execute it to make sure that we practice good testing of our code.
- always create/update docs directory as well as the README.md for any changes on the project documentation.

## Frontend Instructions
- Strictly only modify the code inside app/web and nothing else.
- Make sure that it is mobile ready or just a dynamic webpage.
- Color Theme: #00A6E3 (Primary Brand Blue ), #403D32 (Deep Charcoal Brown), #F4FCFF (Soft Light Background Blue)

## Backend Instructions
- Strictly only modify the code inside app/api and nothing else.

## CDK Instructions
- Strictly only modify the code inside infra/cdk and nothing else.
